# obs-control-tower

AWS Observability Educational Site — a single-page Vite + TypeScript experience for learning the three core pillars of AWS observability: Amazon CloudWatch, CloudWatch Logs Insights, and AWS X-Ray.

## Prerequisites

- Node.js 18+
- npm

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
